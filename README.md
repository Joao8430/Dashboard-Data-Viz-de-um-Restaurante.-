# Dashboard-Data-Viz-de-um-Restaurante.-
🚀 Acabei de finalizar meu projeto de Data Visualization com Power BI! 
Neste dashboard, analisei o desempenho de vendas de restaurantes, focando em:

📊 Receita total
📍 Desempenho por unidade
🍤 Pratos que mais geram faturamento

💡 Insight principal:
Identifiquei quais produtos e unidades têm maior impacto no faturamento, ajudando na tomada de decisão estratégica.
